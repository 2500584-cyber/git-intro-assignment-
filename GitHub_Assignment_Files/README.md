Step-by-Step GitHub Assignment Instructions

1. Unzip this folder to a location on your PC, e.g., C:\Users\YourName\Documents\git-intro-assignment
2. Open Git Bash and navigate to the folder:
   cd /c/Users/YourName/Documents/git-intro-assignment
3. Initialize local git repository:
   git init
4. Stage all files:
   git add .
5. Commit the changes:
   git commit -m "Initial commit with README and sample file"
6. Link to GitHub repository (replace YOUR_USERNAME):
   git remote add origin https://github.com/YOUR_USERNAME/git-intro-assignment.git
7. Rename default branch to main and push to GitHub:
   git branch -M main
   git push -u origin main
8. Verify your files appear on GitHub.
9. Submit your GitHub repository link for the assignment.
